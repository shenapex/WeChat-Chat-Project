# 微信聊天记录项目

**收集与解密微信聊天记录有关的项目、教程等**

本仓库仅用于**个人学习与自用备份**，用于整理和记录互联网上曾公开出现过的、与微信聊天记录结构、数据库格式等相关的资料与项目信息。

本仓库不包含任何破解工具，不提供任何可执行代码，仅记录资料名称、说明或链接，便于个人查阅。

如内容涉及他人项目或文档，其版权均属于原作者。
